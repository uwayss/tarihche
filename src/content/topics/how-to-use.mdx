<Section title="Navigation">

Use the left sidebar to switch topics. Inside a topic, use the right sidebar to jump to any section.

</Section>

<Section title="Progressive Disclosure" defaultOpen={true}>

Sections are collapsible. This supports classroom pacing: open only what you need.

</Section>

<Section title="Annotations" defaultOpen={false}>

Use the annotation toolbar to:

- **Highlight** text selections
- **Pen** draw on top of the content
- **Erase** highlights or pen strokes
- **Clear** to reset the topic annotations

Annotations are stored **client-side** (this browser) per topic.

</Section>
